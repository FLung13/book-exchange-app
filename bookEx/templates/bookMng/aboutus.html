About BookEx 📚

Welcome to BookEx — a mini online bookstore where you can share, discover, and manage books with just a few clicks.

What is BookEx?

BookEx is a web app created as part of the CS 3337 – Software Engineering course. It’s designed to feel like a small bookstore where users can:

  • 📖 Post books with title, cover image, price, and website link
  • 🧑‍💻 Log in and manage only the books they’ve posted
  • 🛒 Add any quantity of a book to the shopping cart
  • ⭐ Rate and favorite books they like
  • 🏷️ Browse books by category like Fiction, Classics, Children’s Books, and more

Why was it built?

This project is a hands-on way to practice real software engineering concepts:

  • ✅ Working with Django models, views, and templates
  • ✅ Handling file uploads for book cover images
  • ✅ Implementing user authentication and permissions
  • ✅ Building features like search, filters, ratings, favorites, and a cart
  • ✅ Keeping the UI clean, responsive, and easy to use

Who can do what?

BookEx includes some basic rules to keep things organized:

  • 👤 Logged-in users can post books and delete their own books.
  • 🛡️ Admin users can delete any book if needed.

Thanks for visiting! 🎉

BookEx isn’t a full commercial bookstore — it’s a learning project that shows what happens behind the scenes of a real web app.
Feel free to explore, post a book, add some favorites, and fill up your cart!
